# Three.js Viewer with Astro

A 3D model viewer built with Three.js and Astro framework.

## Features
- Interactive 3D viewer for glTF models
- Instance selection
- Smooth camera panning with left-click
- Dynamic lighting and shadows

## Setup
1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Usage
- Left-click and drag to pan the camera
- Click on a model to select/deselect it
- Selected models will move forward in the Z-axis

## Technologies Used
- Astro
- Three.js
- TypeScript
